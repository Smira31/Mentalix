# STOIC → MENTALIX: каталог функций

Разбор функционала stoic. по видеозаписи от 24.07.2026.
Каждая функция: что это → зачем она (какую задачу решает) → статус в Mentalix → приоритет.

Приоритеты: **P0** — ядро ежедневного цикла, делаем в первую очередь. **P1** — усиливает retention. **P2** — полировка и глубина. **⚠️** — конфликтует с нашим Product Vision, брать с адаптацией или не брать.

---

## 1. Ежедневный цикл (ядро)

### 1.1. Daily Check-In — многошаговый чек-ин ⭐ главная функция stoic.
**Что это:** цепочка из 4–5 коротких экранов, по одному вопросу на экран:
1. «Как ты?» — настроение по 5-балльной шкале (лица от грустного до сияющего)
2. «Чем был занят?» — сетка тегов-активностей (работа, семья, отношения, друзья, сон, спорт, еда, учёба...), мультивыбор
3. Шкалы-слайдеры — энергия / стресс
4. «Что на уме?» — свободный текст, опционально фото или голос
5. Экран завершения — празднование, серия, «30 000 человек отметились сегодня»

**Зачем:** это входная точка всего приложения. Один тап с главной → 60 секунд → пользователь и выговорился, и накормил аналитику данными (настроение × активности × время = топливо для Трендов). Дробление на шаги по одному вопросу снижает барьер: ответить на один вопрос легко, на анкету из пяти — тяжело.

**В Mentalix:** бэкенд УЖЕ есть (`/checkin` принимает mood, energy, anxiety, focus, note) — но нет интерфейса! Нужен только фронт: пошаговый флоу в стиле stoic.
**Приоритет: P0.** Самый дешёвый большой выигрыш: API готов, рисуем 5 экранов.

### 1.2. Способы ввода записи (FAB-стек)
**Что это:** кнопка «+» раскрывается в стопку: Фото / Голос / Подсказки для записи / Мои шаблоны / Чистая страница.
**Зачем:** прогрессивное раскрытие — сложность спрятана до момента, когда нужна. Голос снижает барьер для тех, кому лень печатать.
**В Mentalix:** голосовой ввод уже есть в ИИ-чатах. Для дневника — нет самого дневника (см. 1.3).
**Приоритет: P1** (после появления дневника).

### 1.3. Дневник с подсказками (Guided Journal)
**Что это:** готовые сценарии записи: «Дневник благодарности», «Feel Better Journal» и т.п. Один вопрос на экран, точки прогресса сверху, пишешь ответ → следующий вопрос → в конце всё склеивается в одну запись.
**Зачем:** «чистый лист» пугает; вопрос-за-вопросом ведёт за руку. Это же строит привычку рефлексии — ядро ценности такого продукта.
**В Mentalix:** нет. Ближайшее — «Считка дня» (цитата) и заметки в курсах.
**Приоритет: P0–P1.** Для нас это «Ритуал рефлексии»: вечерний разбор дня идеально ложится в концепцию ритуалов.

### 1.4. Утренняя подготовка / вечерняя рефлексия
**Что это:** два спец-сценария по времени суток: утром — «какой фокус дня?» (выбор приоритета), вечером — итоги. В Трендах stoic. видно «Your daily focus from Morning Preparation» — утренний выбор потом анализируется.
**Зачем:** привязка к «книжным полкам» дня (начало/конец) — так формируется привычка открывать приложение без пушей. Ровно наш принцип из вижина: Mentalix — первое и последнее приложение дня.
**В Mentalix:** нет. У нас есть временной контекст в приветствии — надо расширить на сценарии.
**Приоритет: P1.**

## 2. Контент и практики

### 2.1. Цитаты (отдельная вкладка в stoic.)
**Что это:** полноэкранные карточки с цитатами, свайп вверх — следующая, сердечко — сохранить, кнопка share — поделиться красивой картинкой.
**Зачем:** лёгкая дофаминовая петля («ещё одну») + share = бесплатный маркетинг: карточка с цитатой уходит в сторис.
**В Mentalix:** «Считка дня» уже есть (1 цитата/день + менеджер цитат). Отдельную вкладку НЕ делаем (у нас 4 таба, вижин запрещает пятую) — вместо этого: тап по «Мысли дня» на главной открывает полноэкранный режим со свайпами.
**Приоритет: P1. ⚠️ адаптируем:** share-картинка — сильный ход для твоего Instagram-проекта: брендированная карточка Mentalix в сторис = органический трафик.

### 2.2. Дыхательная практика
**Что это:** анимированный круг: расширяется — вдох, сжимается — выдох, таймер цикла.
**Зачем:** мгновенная ценность за 60 секунд; «скорая помощь» при стрессе. Даёт поводу открыть приложение днём.
**В Mentalix:** нет. Идеально ложится в «Практики» рядом с Фокусом.
**Приоритет: P1** — дёшево сделать (одна анимация), заметная ценность.

### 2.3. Тема недели (Weekly Theme)
**Что это:** 7-дневная серия размышлений на одну тему («о меньших усилиях», день 5 из 7), каждый день — новый текст + кнопка «Reflect» (написать мысли). Частично под замком (premium).
**Зачем:** серийный контент = причина возвращаться каждый день; лёгкая монетизация.
**В Mentalix:** нет как формата, НО у тебя есть Библиотека — тема недели это по сути мини-курс, порезанный на 7 дней.
**Приоритет: P2**, но это готовый формат для платной Библиотеки: бесплатные дни 1–2, остальное по подписке. И это же — твой контент-конвейер: материалы для тем недели = твои 3П-конспекты книг.

### 2.4. Аффирмация дня + «Персонализировать»
**Что это:** карточка с установкой («Я доверяю тому, что всё складывается...»), кнопки «Journal About It» и «Personalize» (выбор тем аффирмаций).
**Зачем:** пассивная ценность для тех, кто сегодня не готов писать; персонализация даёт ощущение «приложение знает меня».
**В Mentalix:** частично — «Считка дня». Добавить кнопку «Обсудить с наставником» (вместо Journal About It — сразу мостик к нашим ИИ).
**Приоритет: P2.**

## 3. Память и аналитика

### 3.1. История
**Что это:** лента всех записей по дням («Сегодня, 24 июля» → чек-ин 18:02, достижение, запись), фильтры по типу, поиск, переключение день/неделя/месяц.
**Зачем:** дневник без перечитывания — просто база данных. История = «я вижу свой путь позади» — усиливает идентичность.
**В Mentalix:** нет отдельного экрана (логи ритуалов/аскез есть в БД, но пользователь их не видит).
**Приоритет: P1.** Вписать в «Путь»: вкладка/секция «История» внутри экрана Пути — прошлое и будущее в одном месте, это сильнее, чем у stoic.

### 3.2. Тренды
**Что это:** график настроения по неделе, топ активностей (пончик-диаграмма из тегов чек-ина), инсайты. Пустые состояния активные: «нет данных — отметь настроение прямо сейчас» с рядом смайликов.
**Зачем:** replay-ценность данных: «в дни со спортом настроение выше» — инсайт, ради которого продолжают чек-иниться.
**В Mentalix:** Аналитика уже есть (проценты, корреляции, график недели). Донастроить: график настроения из чек-инов + топ-активности, активные пустые состояния.
**Приоритет: P0** (доработка существующего) — но только после чек-ина (1.1), без него нет данных.

### 3.3. Достижения
**Что это:** ачивки за вехи («First Steps — первый чек-ин») с иллюстрациями, показываются в истории и на экране завершения.
**Зачем:** микро-награда в момент действия; первая ачивка в первую минуту = «у меня уже что-то есть, жалко бросать».
**В Mentalix:** нет. ⚠️ по вижину: без геймификации-ради-геймификации — ачивки привязываем к Пути (вехи 10%/25%/50%), не к счётчикам.
**Приоритет: P2.**

### 3.4. Серия (streak) с заморозками
**Что это:** огонёк с числом дней подряд, заморозки прощают пропуск.
**Зачем:** loss aversion — терять жалко. Заморозки у Duolingo снизили отток на 21%.
**В Mentalix:** УЖЕ ЕСТЬ на бэкенде (серии с заморозками в ритуалах). ⚠️ вижин прямо запрещает streak как давление: показываем «непрерывность Пути» (дней с открытием из 30), не «серию, которую страшно потерять». Огонёк в шапке — только после первого выполненного действия, не «0» с порога (ошибка stoic., мы её у себя уже видели).
**Приоритет: P1** — вывести бережно, в терминах Пути.

## 4. Обвязка

### 4.1. Напоминания по времени
**В Mentalix:** есть (настройки + бот). Донастроить контекстные: утро/вечер/восстановление — не спам. **P2.**

### 4.2. Социальное доказательство
**Что это:** «30 493 человека отметились сегодня» под чек-ином.
**Зачем:** снимает одиночество, подталкивает присоединиться.
**В Mentalix:** нет. Легко: счётчик чек-инов за день из БД. Пока пользователей мало — честная формулировка («вместе с тобой сегодня: 12») или скрыть до порога. **P2.**

### 4.3. Пейволл на контент
**Что это:** замочки на темах недели/шаблонах, экран подписки.
**В Mentalix:** план уже есть — Библиотека через Telegram Stars, бесплатная вводная часть. **P1** (после ядра).

### 4.4. Редактор шаблонов («Your Templates»)
**Что это:** пользователь собирает свой сценарий дневника из вопросов.
**В Mentalix:** ⚠️ P3, отложить — это фича для продвинутых, сначала дать готовые сценарии.

## Порядок внедрения (предложение)

**Этап 1 — «Ежедневный цикл» (ядро):**
1. Чек-ин флоу (1.1) — фронт к готовому API
2. Тренды на данных чек-ина (3.2)
3. Экран завершения с празднованием + серия в терминах Пути (3.4)

**Этап 2 — «Глубина»:**
4. Дневник с подсказками: вечерняя рефлексия (1.3 + 1.4)
5. История внутри Пути (3.1)
6. Полноэкранные цитаты со свайпом + share-картинка (2.1)

**Этап 3 — «Контент и деньги»:**
7. Дыхание (2.2)
8. Библиотека: платные материалы + формат «Тема недели» (2.3 + 4.3)
9. Достижения по вехам Пути (3.3)

  (пол/возраст реальной аудитории) и скорректировать тон по фактам.

## Что из stoic. НЕ берём (по Product Vision)

- ❌ Пятую вкладку (Quotes) — цитаты живут на главной
- ❌ Streak как давление («не потеряй серию!») — только непрерывность Пути
- ❌ Огонёк «0» с первой секунды — появляется после первого действия
- ❌ Редактор шаблонов в MVP — scope creep
- ❌ Отдельную ленту медитаций — мы не wellness-платформа, мы OS роста
