# Mentalix — Roadmap

Трекер привычек, целей и состояния с ИИ-ассистентом. Telegram Mini App.

## Стек
- **Фронт:** React + Vite + Tailwind, деплой на Vercel (репозиторий Mentalix)
- **Бот:** aiogram (папка bot/), Railway — сервис accurate-expression
- **Backend:** FastAPI (папка backend/), Railway — сервис mentalix-bot
- **База:** PostgreSQL на Railway

## Готово

### Дизайн
- Редизайн 5 вкладок: Сегодня / Мой путь / Аналитика / Mentalix / Профиль
- Палитра: emerald-deep/emerald/emerald-light, sage, mint, cognac, gold, cream
- Шрифты: Fraunces (заголовки) + Manrope (текст)
- Нижний навбар с круглыми кнопками (активная — золотой круг)
- Кольцевые шкалы, wireframe-горы, elevation-графики
- Карточка состояния «Как ты себя чувствуешь» с эмодзи-шкалой, сворачивается после заполнения
- Приветствие по времени суток
- Иллюстративные пустые состояния

### Функциональность
- Привычки: создание, удаление, отметка (минимум/оптимум), детальный экран
- Цели: создание, удаление, детальный экран, привязка привычек
- Чек-ины: настроение / энергия / тревога / фокус
- Голосовой ввод в чате
- ИИ-ассистент Mentalix (GigaChat) с доступом к данным пользователя, /summary

### Retention
- Ежедневные напоминания (настраиваемое время)
- Еженедельный дайджест (воскресенье 19:00 UTC)
- Камбэк-уведомления (после 3 дней неактивности)
- Стрики с защитой (заморозки: +1 за 7 дней серии, максимум 2)
- Живые тексты уведомлений с характером

### Онбординг
- Пресеты привычек (9 шаблонов) + умное распознавание иконок
- Экраны первого запуска для новых пользователей

### Аналитика
- Инсайты и графики (настроение/тревога, активность за неделю)
- Корреляции: связь привычек с самочувствием

### Juice / ощущение
- Момент награды при отметке: галочка-«поп», свечение карточки, подскок серии, усиленный haptic

## Решено не делать
- **Настройка тона ассистента** — единый honest-тон сильнее размывания на режимы
- **Экспорт CSV** — аналитика в приложении нагляднее, отдача файла капризна на Railway

## Идеи на будущее
- Проактивные инсайты от бота (сам присылает наблюдение раз в неделю)
- PDF-отчёт (если появится потребность)
- Празднование крупных вех серии (7/21/50 дней)
